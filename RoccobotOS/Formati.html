Fattore = 'ingrandimento': definisce il numero effettivo di pixel renderizzati in base ai dp del modello
Classe = sigla che categorizza gli schermi dei dispositivi in base alla densità (DPI)
DPI (Dots Per Inch) = densità dei pixel fisici dello schermo (misurata in pixel per pollice)
px = pixel (elementi che compongono lo schermo reale): orizzontali × verticali
Proporzione (es: 9:16) = rapporto larghezza / altezza che definisce la forma dello schermo
dp (o DiP) = Density-independent Pixel: definiscono la risoluzione base di un dispositivo (pixel a 1x)

A 160 DPI ('baseline') pixel e dp coincidono (1 px = 1 dp):

px = dp * (DPI / 160)
